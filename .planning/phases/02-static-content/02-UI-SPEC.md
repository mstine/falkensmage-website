## Spacing Scale

All multiples of 4. Declared as px equivalents of the rem values in use across the codebase.

| Token | Value | CSS Equivalent | Usage |
|-------|-------|----------------|-------|
| xs | 4px | 0.25rem | Icon-to-label gap, internal padding |
| sm | 8px | 0.5rem | Tight inline spacing, subtle gaps |
| md | 16px | 1rem | Default element spacing, paragraph gaps |
| xl | 24px | 1.5rem | Section horizontal padding (mobile), social card gap, section content gap |
| 2xl | 32px | 2rem | Section horizontal padding (tablet+), major element breaks |
| 3xl | 48px | 3rem | Section vertical padding (mobile) |
| 4xl | 64px | 4rem | Section vertical padding (tablet and desktop) |

**Exceptions:**
- Social card padding: 12px vertical × 16px horizontal (`0.75rem 1rem`) — visual breathing room — touch target guaranteed by min-height rule
- CTA button padding: 12px vertical × 32px horizontal (`0.75rem 2rem`) — larger horizontal to give the Radiant Core gradient room to breathe
- Touch targets: 44px minimum height on all `.social-card` elements (WCAG 2.5.5 / SOCIAL-03)
- Glow bleed: `inset: -20%` on `.glow-ambient::before` and `inset: -4px` on `.glow-interactive::before` — decorative, not spacing system
- Section transition band: `height: 4rem` / `top: -2rem` on `.section + .section::before` — structural, not spacing system

*Source: themes/arcaeon/assets/css/main.css (Phase 1 established values)*

## Typography

Four-tier system: Display > H1 > H2/Tagline > Body/Label. Phase 2 governs two weights in active use for non-heading content (400 body, 600 CTA/label). Display (800) and H1 (700) weights are Phase 1 inherited values — not new Phase 2 declarations.

| Role | Font | Size | Weight | Line Height | Usage |
|------|------|------|--------|-------------|-------|
| Display | Cinzel Variable | `clamp(2.5rem, 8vw, 4rem)` | 800 (Phase 1) | 1.1 | "Falken's Mage" hero logotype — `.display-text` class |
| Heading 1 | Cinzel Variable | 2.074rem (33px) | 700 (Phase 1) | 1.2 | Section headings (Identity+CTA section header if used) |
| H2 / Tagline | Cinzel Variable or Space Grotesk Variable | 1.728rem (28px) | 600 | 1.3 | Sub-headings within sections; also used for the hero tagline slot below "Falken's Mage" — `.hero-tagline` |
| Body / Label | Space Grotesk Variable | 1rem (16px) | 400 (body) / 600 (label) | 1.6 (body) / 1.2 (label) | Identity statement, CTA description, footer text (400); social card labels, CTA button text, nav labels (600) |

**Phase 2 weight declarations (2 weights):**
- 400 — Body text (identity statement, CTA description, footer text)
- 600 — Labels and sub-headings (social card labels, CTA button, H2/Tagline tier)

**Phase 1 inherited weights (not re-declared in Phase 2):**
- 700 — H1 heading (section headings, established in Phase 1 `h1` rule)
- 800 — Display (logotype at scale, established in Phase 1 `.display-text` rule)

**Notes:**
- Hero tagline uses the H2/Tagline tier (1.728rem, weight 600) — sufficient typographic separation from the Display logotype above it, no separate `--text-h3` needed
- Body line-height 1.6 (established in Phase 1 `body {}` rule — slightly above the recommended 1.5 for Space Grotesk readability on dark backgrounds)
- Heading line-height 1.2 matches the Phase 1 `h1, h2, h3 {}` rule
- Display line-height 1.1 is tighter than standard — appropriate for the single-line logotype at large scale
- All heading elements use Cinzel; all non-heading elements use Space Grotesk — no mixing

*Source: themes/arcaeon/assets/css/main.css — `:root` typography tokens*

## Color

| Secondary (30%) | `--arcaeon-deep-indigo` | `#0a0f3c` | Identity+CTA section bg, Footer bg (`.section-depth`) |
| Transition | `--arcaeon-midnight-blue` | `#111a6b` | Section-to-section gradient bands only |
| Accent (10%) | `--arcaeon-electric-violet` | `#7a2cff` | See reserved list below |

### Accent Reserved For (10% — Electric Violet)

Electric Violet is **decoration and glow only**. It fails WCAG AA for body text on both primary backgrounds (established Phase 1 constraint, documented in CSS).

Reserved for:
1. Hero image ambient glow pulse (`.glow-ambient::before`)
2. Social card hover glow fill (`.glow-interactive::before`)
3. Social card border at rest (`rgba(122, 44, 255, 0.35)`)
4. Social card border on hover (full `--arcaeon-electric-violet`)
5. Sigil arc decorative shapes at section boundaries (`--arcaeon-electric-violet`, opacity 0.3)

**Never:** body text, identity statement, tagline, CTA description, footer text, social card labels.

Reading flow: professional cluster (1–4) then personal/creative cluster (5–8).

## Interaction States

### Social Cards

| State | Visual |
|-------|--------|
| Rest | Ghost card — Electric Violet border at `rgba(122, 44, 255, 0.35)`, Solar White label, neutral icon |
| Hover / Focus-visible | Border color → `--arcaeon-electric-violet` (full), label → `--arcaeon-ion-glow`, icon → `currentColor` (inherits Ion Glow), `.glow-interactive::before` opacity → 0.6 |
| Transition | `border-color 0.25s ease`, `color 0.25s ease`, glow opacity `0.25s ease` |
| Reduced motion | `transition: none` on glow; border/color transitions may remain (non-animation) |

### CTA Button

| State | Visual |
|-------|--------|
| Rest | Radiant Core gradient (Fusion Gold → Ignition Orange), Void Purple text |
| Hover / Focus-visible | `.glow-radiant-core::before` opacity → 0.7 (warm Fusion Gold radial) |
| Transition | `opacity 0.3s ease` on glow |
| Reduced motion | `transition: none` on glow pseudo-element |

### Ambient Hero Glow

| State | Visual |
|-------|--------|
| Active | `ambient-pulse` keyframe: opacity oscillates 0.2 → 0.5 → 0.2, 5s infinite |
| Reduced motion | Animation stopped; glow locked at opacity 0.3 (static, visible but not moving) |

### Links (inline, footer)

| State | Visual |
|-------|--------|
| Rest | Ion Glow (`#5be7ff`) |
| Hover | Neon Magenta (`#ff3cac`) |

## Accessibility Contract

| Requirement | Implementation | Standard |
|-------------|----------------|----------|
| Touch targets | `min-height: 44px` on all `.social-card` elements | WCAG 2.5.5 (SOCIAL-03) |
| Social link labels | `aria-label="{{ .name }}"` on every `<a>` | WCAG 4.1.2 (SOCIAL-05) |
| Hero image alt | Descriptive scene alt text on `<img>` | WCAG 1.1.1 (A11Y-03) |
| Lemniscate mark | `aria-hidden="true"` — decorative, no semantic value | WCAG 1.1.1 |
| Reduced motion | `@media (prefers-reduced-motion: reduce)` suppresses all animation | WCAG 2.3.3 (A11Y-04) |
| Color contrast | Solar White / Ion Glow on dark backgrounds — AA compliant. Electric Violet never used for text (established Phase 1 rule) | WCAG 1.4.3 (A11Y-02) |
| Semantic HTML | `<header>`, `<main>`, `<section>`, `<footer>` — established in baseof.html | WCAG 1.3.1 (A11Y-01) |
| External link security | `target="_blank" rel="noopener noreferrer"` on all social links | Security hygiene (not WCAG) |
| Mailto link | No `target="_blank"` — native OS protocol handler, not a URL | Standard HTML |
