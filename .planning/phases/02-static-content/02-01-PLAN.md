```html
    <picture>
      <source media="(min-width: 768px)" type="image/webp" srcset="{{ $desktopWebP.RelPermalink }}">
      <source media="(min-width: 768px)" type="image/jpeg" srcset="{{ $desktopResize.RelPermalink }}">
      <source type="image/webp" srcset="{{ $mobileWebP.RelPermalink }}">
      <img src="{{ $mobileFill.RelPermalink }}"
           alt="Matt Stine as the Magus — laptop open, wand raised, tarot cards in hand, raven on the shelf, lemniscate overhead"
           width="{{ $mobileFill.Width }}"
           height="{{ $mobileFill.Height }}"
           loading="eager"
           class="hero-image">
    </picture>
  </div>
  <div class="section-content hero-text">
    <h1 class="display-text">{{ $.Title | default "Falken's Mage" }}</h1>
    {{ with $.Params.tagline }}<p class="hero-tagline">{{ . }}</p>{{ end }}
  </div>
</section>
{{ end }}
```

Key details:
- Image above, text below on all viewports (stacked layout per D-01, D-04)
- `.glow-ambient` wraps the image container (per D-03) — Electric Violet bleeds from behind the Magus
- `.Fill "750x580 Center"` crops the 1424x752 source for mobile 2x retina (per D-02)
- `.Resize "1200x q85"` preserves full landscape for desktop
- `.Process "webp"` for format conversion (NOT `.Format "webp"` which errors in Hugo 0.160.1)
- `loading="eager"` — hero is above fold, must not lazy-load
- `{{ with $img }}` guard prevents template error if image missing
