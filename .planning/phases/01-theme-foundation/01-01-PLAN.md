    **Step 1 — Install Hugo Extended via Homebrew:**
    ```
    brew install hugo

    ```
    Verify output contains "extended". If it shows standard edition, install from GitHub releases instead.

    **Step 2 — Initialize Hugo site in current directory:**
    The project root already has CLAUDE.md and SPEC.md. Run:
    ```
    hugo new site . --force
    ```
    The `--force` flag allows init in a non-empty directory.

    **Step 3 — Configure hugo.toml (per INFRA-02, D-01):**
    Replace the generated hugo.toml with:
    ```toml
    baseURL = "https://falkensmage.com/"
    languageCode = "en-us"
    title = "Falken's Mage"
    theme = "arcaeon"

    [params]
      author = "Matt Stine"
      description = "Falken's Mage - Matt Stine"

    [params.colors]
      electricViolet = "#7a2cff"
      neonMagenta = "#ff3cac"
      plasmaPink = "#ff5fd2"
      electricBlue = "#2fd3ff"
      solarWhite = "#fff4e6"
      fusionGold = "#ffb347"
      ignitionOrange = "#ff7a18"
      deepIndigo = "#0a0f3c"
      midnightBlue = "#111a6b"
      voidPurple = "#1a0f2e"
      ionGlow = "#5be7ff"

    [caches]
      [caches.getresource]
        dir = ":cacheDir/:project"
        maxAge = "1h"
    ```

    **Step 4 — Install font packages and copy WOFF2 files:**
    ```
    npm init -y
    npm install @fontsource-variable/cinzel @fontsource-variable/space-grotesk
    ```
    Then inspect actual filenames:
    ```
    ls node_modules/@fontsource-variable/cinzel/files/
    ls node_modules/@fontsource-variable/space-grotesk/files/
    ```
    Create target directory and copy the Latin variable WOFF2 files (look for files matching `*latin*wght*woff2` pattern):
    ```
    mkdir -p themes/arcaeon/assets/fonts
    cp node_modules/@fontsource-variable/cinzel/files/cinzel-latin-wght-normal.woff2 themes/arcaeon/assets/fonts/
    cp node_modules/@fontsource-variable/space-grotesk/files/space-grotesk-latin-wght-normal.woff2 themes/arcaeon/assets/fonts/
    ```
    Adjust filenames if the actual names differ from this pattern. The important thing is to get the variable-weight Latin WOFF2 files.

    **Step 5 — Create theme directory scaffold (THEME-07):**
    ```
    mkdir -p themes/arcaeon/layouts/_default
    mkdir -p themes/arcaeon/layouts/partials/sections
    mkdir -p themes/arcaeon/layouts/partials/head
    mkdir -p themes/arcaeon/assets/css
    mkdir -p themes/arcaeon/static
    ```

    **Step 6 — Create baseof.html** at `themes/arcaeon/layouts/_default/baseof.html`:
    ```html
    <!DOCTYPE html>
    <html lang="{{ site.Language.Lang }}">
    <head>
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <title>{{ if .IsHome }}{{ site.Title }}{{ else }}{{ .Title }} | {{ site.Title }}{{ end }}</title>

      {{/* Font preloads — prevent FOIT */}}
      {{ $cinzel := resources.Get "fonts/cinzel-latin-wght-normal.woff2" | fingerprint }}
      <link rel="preload" href="{{ $cinzel.RelPermalink }}" as="font" type="font/woff2" crossorigin>
      {{ $spaceGrotesk := resources.Get "fonts/space-grotesk-latin-wght-normal.woff2" | fingerprint }}
      <link rel="preload" href="{{ $spaceGrotesk.RelPermalink }}" as="font" type="font/woff2" crossorigin>

      {{/* CSS via Hugo css.Build pipeline */}}
      {{ $css := resources.Get "css/main.css" | css.Build | minify | fingerprint }}
      <link rel="stylesheet" href="{{ $css.RelPermalink }}">
    </head>
    <body>
      {{ block "main" . }}{{ end }}
    </body>
    </html>
    ```
    NOTE: The font filenames in the preload links MUST match the actual filenames copied in Step 4. Verify and adjust if needed.

    **Step 7 — Create index.html** at `themes/arcaeon/layouts/index.html`:
    ```html
    {{ define "main" }}
    <main>
      {{ .Content }}
    </main>
    {{ end }}
    ```

    **Step 8 — Create content/_index.md:**
    ```markdown
    ---

    ---
    ```

    **Step 9 — Update .gitignore:**
    Add these entries to `.gitignore` (Hugo generates one, append to it):
    ```
    node_modules/
    resources/_gen/
    .hugo_build.lock
    public/
    ```

    **Step 10 — Verify Hugo builds:**
    ```
    hugo server --buildDrafts
    ```
    Confirm no errors. Kill the server after confirming.

       ============================================ */

      --arcaeon-electric-violet: #7a2cff;
      --arcaeon-neon-magenta: #ff3cac;
      --arcaeon-plasma-pink: #ff5fd2;
      --arcaeon-electric-blue: #2fd3ff;

      /* Focal Energy (Radiant Core) */
      --arcaeon-solar-white: #fff4e6;
      --arcaeon-fusion-gold: #ffb347;
      --arcaeon-ignition-orange: #ff7a18;

      /* Cosmic Depth */
      --arcaeon-deep-indigo: #0a0f3c;
      --arcaeon-midnight-blue: #111a6b;
      --arcaeon-void-purple: #1a0f2e;

      /* Energy Accents */
      --arcaeon-ion-glow: #5be7ff;
    }

    /* ============================================
       TOKENS — Semantic Aliases
       Usage-intent layer on top of palette truth (D-01)
       ============================================ */

    :root {
      --color-bg: var(--arcaeon-void-purple);
      --color-bg-alt: var(--arcaeon-deep-indigo);
      --color-bg-transition: var(--arcaeon-midnight-blue);
      --color-text: var(--arcaeon-solar-white);
      --color-text-accent: var(--arcaeon-ion-glow);
      --color-accent: var(--arcaeon-electric-violet);
      --color-accent-warm: var(--arcaeon-fusion-gold);
      --color-hover: var(--arcaeon-neon-magenta);
      --color-cta-start: var(--arcaeon-fusion-gold);
      --color-cta-end: var(--arcaeon-ignition-orange);
    }

    /* ============================================
       TOKENS — Triad Rule Section Overrides (D-02)
       Each section gets scoped --section-purple, --section-blue, --section-warm
       Convention: every section partial uses one of these classes
       ============================================ */

    .section-void {
      --section-purple: var(--arcaeon-void-purple);
      --section-blue: var(--arcaeon-electric-blue);
      --section-warm: var(--arcaeon-fusion-gold);
      background: var(--section-purple);
    }

    .section-depth {
      --section-purple: var(--arcaeon-deep-indigo);
      --section-blue: var(--arcaeon-ion-glow);
      --section-warm: var(--arcaeon-ignition-orange);
      background: var(--section-purple);
    }

    /* ============================================
       RESET
       Minimal reset — box-sizing, margin, padding, smooth scroll
       ============================================ */

    *,
    *::before,
    *::after {
      box-sizing: border-box;
      margin: 0;
      padding: 0;
    }

    html {
      scroll-behavior: smooth;
    }

    @media (prefers-reduced-motion: reduce) {
      html {
        scroll-behavior: auto;
      }
    }

    body {
      background-color: var(--color-bg);
      color: var(--color-text);
      min-height: 100vh;
      -webkit-font-smoothing: antialiased;
      -moz-osx-font-smoothing: grayscale;
    }

    img {
      max-width: 100%;
      height: auto;
      display: block;
