    However, Hugo css.Build processes CSS from the assets directory. The `url()` paths in CSS processed by css.Build should use paths relative to the CSS file location within the assets directory. So from `assets/css/main.css`, the path to fonts is `../fonts/{filename}.woff2`.

    ```css
    /* ============================================
       TYPOGRAPHY — Font Face Declarations
       Self-hosted variable fonts via @fontsource-variable
       font-display: swap prevents FOIT (D-05)
       ============================================ */

    @font-face {
      font-family: 'Cinzel Variable';
      src: url('../fonts/cinzel-latin-wght-normal.woff2') format('woff2');
      font-weight: 400 900;
      font-style: normal;
      font-display: swap;
    }

    @font-face {
      font-family: 'Space Grotesk Variable';
      src: url('../fonts/space-grotesk-latin-wght-normal.woff2') format('woff2');
      font-weight: 300 700;
      font-style: normal;
      font-display: swap;
    }

    /* ============================================
       TYPOGRAPHY — Type Scale (D-04, Minor Third 1.2 ratio)
       Base: 1rem (16px)
       Display treatment outside scale: clamp(2.5rem, 8vw, 4rem)
       ============================================ */

    :root {
      --font-display: 'Cinzel Variable', 'Times New Roman', serif;
      --font-heading: 'Cinzel Variable', 'Times New Roman', serif;
      --font-body: 'Space Grotesk Variable', system-ui, sans-serif;

      /* Minor Third scale (D-04) */
      --text-base: 1rem;
      --text-h3: 1.44rem;
      --text-h2: 1.728rem;
      --text-h1: 2.074rem;
      --text-display: clamp(2.5rem, 8vw, 4rem);

      /* Cinzel weight progression (D-06) */
      --weight-display: 800;
      --weight-h1: 700;
      --weight-h2: 600;
      --weight-h3: 500;
    }

    /* ============================================
       TYPOGRAPHY — Element Styles
       Cinzel = headings (D-05), Space Grotesk = everything else
       ============================================ */

    body {
      font-family: var(--font-body);
      font-size: var(--text-base);
      font-weight: 400;
      line-height: 1.6;
    }

    h1, h2, h3 {
      font-family: var(--font-heading);
      line-height: 1.2;
      color: var(--color-text);
    }

    h1 {
      font-size: var(--text-h1);
      font-weight: var(--weight-h1);
    }

    h2 {
      font-size: var(--text-h2);
      font-weight: var(--weight-h2);
    }

    h3 {
      font-size: var(--text-h3);
      font-weight: var(--weight-h3);
    }

    .display-text {
      font-family: var(--font-display);
      font-size: var(--text-display);
      font-weight: var(--weight-display);
      line-height: 1.1;
    }

    a {
      color: var(--color-text-accent);
      text-decoration: none;
    }

    a:hover {
      color: var(--color-hover);
    }
    ```

    ```css
    /* ============================================
       GLOW SYSTEM — Ambient Pulse (D-07, D-09, D-10)
       Slow breathing radial glow for hero image / decorative elements
       Color: Electric Violet (#7a2cff) per D-09
       Timing: 5s ease-in-out cycle per D-10 (4-6s range)
       GPU-composited: ::before + opacity + will-change (D-08)
       ============================================ */

    .glow-ambient {
      position: relative;
    }

    .glow-ambient::before {
      content: '';
      position: absolute;
      inset: -20%;
      background: radial-gradient(
        circle,
        var(--arcaeon-electric-violet) 0%,
        transparent 70%
      );
      opacity: 0.4;
      animation: ambient-pulse 5s ease-in-out infinite;
      pointer-events: none;
      will-change: opacity;
      z-index: -1;
    }

    @keyframes ambient-pulse {
      0%, 100% { opacity: 0.2; }
      50% { opacity: 0.5; }
    }

    @media (prefers-reduced-motion: reduce) {
      .glow-ambient::before {
        animation: none;
        opacity: 0.3;
      }
    }

    /* ============================================
       GLOW SYSTEM — Interactive Glow (D-07)

      will-change: opacity;
    ```
